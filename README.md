# First-Repo

git 연습을 위한 나의 첫 레포

[See Demo](https://www.google.com)

## Prerequisites

- git
- python^3.13
- pandas==1.0.0

## How to start(Quickstart)

```shell
$ git clone {addr}
$ cd first-repo
$ python main.py
```

## Taylor series

$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n$
